# Archive — early pilot (superseded)

`dashboard.html` is an **early exploratory version** of this project and is **not** the current
methodology. It was built before the pipeline settled on the TNBC logic-gating workflow.

It contains three tabs:

- **① Concept Map** — the (WHERE × WHAT) grid framing. *Still conceptually valid* — the current
  version lives in [`../figures/concept_map_en.png`](../figures/concept_map_en.png) and in the deck.
- **② Data Sources** — an early data-source catalog.
- **③ Lab Notebook** — per-grid-cell protocols (P02–P05, PDAC-oriented). **Superseded.** These
  cell-by-cell protocols were an initial pilot approach and are **no longer followed**; the project
  now uses the single TNBC surface + protease logic-gating pipeline documented in
  [`../README.md`](../README.md) and [`../docs/methods.md`](../docs/methods.md).

Kept here for provenance / to show how the project evolved. For the current work, start at the
repository root.
